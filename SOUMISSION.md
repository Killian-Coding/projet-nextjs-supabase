# Projet 3 - Exploration Technologique avec IA

## Informations de l'équipe
- **Membres:** ABDESSALAM et Killian KILLIAN
- **Date de soumission:** 8 decembre 2025

## Sujet choisi

### Technologies principales
- **Next.js 15** - Framework React avec App Router et Server Components
- **Supabase** - Backend-as-a-Service (PostgreSQL + Realtime)
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling moderne

### Description du projet
Application collaborative de gestion de tâches en temps réel démontrant:

1. **Next.js 15 Server Components & Server Actions**
   - Architecture serveur/client moderne
   - Server Actions remplaçant les API routes
   - Optimisation performances avec SSR

2. **Supabase Realtime**
   - Mise à jour instantanée multi-utilisateurs
   - PostgreSQL avec Row Level Security
   - WebSockets pour temps réel

### Objectifs d'apprentissage
- Maîtriser les Server Components (stable dans Next.js 15)
- Architecture moderne sans API REST traditionnelle
- Sécurité avec Row Level Security
- Utilisation de l'IA pour apprentissage accéléré

### Répartition des tâches
- **ABDESSALAM:** Backend, Server Actions, Database, Supabase
- **Killian:** Frontend, UI/UX, React Components, Realtime
